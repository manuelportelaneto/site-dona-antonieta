# 🎭 Site Dona Antonieta - Portfolio de Entretenimento

Um site one-page vibrante e carismático para o artista **Gilberto Lima** e sua aclamada personagem **Dona Antonieta**, focado na contratação de shows e serviços de entretenimento.

## 🌟 Recursos Implementados

### ✅ Funcionalidades Atuais

#### 🎨 **Design e Interface**
- Layout one-page responsivo e moderno
- Paleta de cores inspirada na bandeira italiana (verde, branco, vermelho, amarelo)
- Tipografia carismática com fontes Google Fonts (Pacifico, Fredoka One, Inter)
- Animações suaves e efeitos visuais atrativos
- Gradientes e sombras para profundidade visual

#### 🧭 **Navegação**
- Navegação fixa com transparência dinâmica
- Menu hambúrguer responsivo para dispositivos móveis
- Scroll suave entre seções
- Botão flutuante do WhatsApp

#### 🎪 **Seções do Site**

1. **Hero Section**
   - Título impactante: "Chega de evento morno! A Dona Antonieta chegou para esquentar a festa!"
   - Estatísticas em destaque (10.000+ pessoas, 44+ anos de experiência)
   - Call-to-action principal
   - Fundo com gradiente das cores da Itália

2. **"Piacere, Sono Antonieta!"**
   - Carrossel de imagens da personagem
   - Apresentação em primeira pessoa da Dona Antonieta
   - Destaques dos principais diferenciais
   - Auto-play e controles manuais

3. **Sobre Gilberto Lima**
   - Biografia completa de 44 anos de carreira
   - Experiências em teatro, TV e cinema
   - Badge de experiência animado
   - Citação inspiracional

4. **Serviços**
   - 3 cards de serviços principais:
     - Show da Nonna (eventos corporativos e sociais)
     - Direção artística para escolas
     - Mestre de cerimônias e animação
   - Ícones e descrições detalhadas

5. **Credibilidade**
   - Lista de clientes e parceiros
   - Eventos em destaque
   - Prova social com organizações respeitadas

6. **Galeria**
   - Grid responsivo de fotos
   - Lightbox para visualização ampliada
   - Navegação por teclado e clique

7. **Contato**
   - Formulário funcional com validação
   - Links diretos para redes sociais
   - Integração com WhatsApp
   - Informações de contato completas

#### 🔧 **Funcionalidades Técnicas**
- Carrossel automático com controles manuais
- Lightbox para galeria de imagens
- Formulário com validação em tempo real
- Sistema de notificações
- Animações de scroll com Intersection Observer
- Placeholders automáticos para imagens quebradas
- Efeitos de hover e transições suaves

#### 📱 **Responsividade**
- Design mobile-first
- Breakpoints para tablet e desktop
- Grid CSS responsivo
- Imagens otimizadas com lazy loading
